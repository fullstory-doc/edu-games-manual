KOffice

Die KOffice Suite beinhaltet folgende Applikationen:

Produktive Applikationen
  • KWord - Ein rahmenbasiertes Textverabeitungsprogramm das in zwei Modi arbeitet: Seiten- oder Layoutorientiert
  • KSpread - Ein mächtiges Programm zur Tabellenkalkulation
  • KPresenter - Ein vollwertiges Präsentationsprogramm.
  • Kexi - Eine integrierte Umgebung zum kreiren von Datenbanken und Datenbankapplikationen
Kreative Applikationen
  • Kivio - Eine im Visio®-stil gehaltene Flussdiagrammanwendung
  • Karbon14 - Eine Anwendung zur Vektorzeichnung
  • Krita - Eine Anwendung zur Bildmanipulation die Mehrschichtigkeit beherrscht
Management Applikationen
  • KPlato - Eine Tool zum Planen und Leiten von Projekten (Projektmanagementtool)
Unterstützende Applikationen
  • KChart - Ein Tool zum zeichnen von Graphen und Diagrammen
  • KFormula - Ein mächtiger Formeleditor.
  • Kugar - Ein Tool zum Generieren von Geschäftsberichten
Kompatibilitätsinformation
  • Unterstützte Dateiformate - Informationen über Kompatibilität mit Dateien die von anderen Programmen oder Applikationen erzeugt wurden.

Eine umfassende Beschreibung ist auf der offiziellen Seite der KDE Office Suite

Openoffice

OpenOffice.org 2 - Produkt Beschreibung

Kompatibel mit anderen grossen Office Suits.

  • Schreiben: Writer – ein Textverabeitungsprogramm zum Schreiben von schnellen Briefen bis hin zu ganzen Büchern.
  • Tabellenkalkulation: Calc – eine mächtige Tabellenkalkulation mit all den Tools zum berechnen, analysieren und präsentieren von Daten in numerischen Reporten oder Grafiken.
  • Präsentation: Impress – der schnellste und beste Weg effektive multimediale Präsentationen zu erstellen.
  • Vektordarstellung: Draw – produziert alles von simplen Diagrammen bis zu dynamischen 3D Illustrationen.
  • Datenbank: Base – Nahtloses Bearbeiten von Datenbanken: Kreiren und Ändern von Tabellen, Formularen, Queries und Reporten - alles aus OpenOffice.org
  • Erzeugen mathematischer Funktionen: Math – Erzeugen von mathematischen Ausdrücken durch eine GUI oder direkt im Funktioneneditor.

Gnumeric

Die Gnumeric Tabellenkalkulation ist Teil der GNOME Desktopumgebung: ein Projekt das ein freie, benutzerfreundliche Desktopumgebung herstellt.

Das Ziel Gnumerics ist die beste mögliche Tabellenkalkulation zu sein, es wird nicht versucht ein bestehendes Programm nachzubilden. Dennoch kann Gnumeric Dateien anderer Tabellenkalkutionen öffnen. Das Aussehen von Gnumeric kann angepasst werden um Umstiegskosten zu senken.

AbiWord

Ein übersetztes Zitat von der AbiWord Webiste: "Wie viele Open Source Projekte began AbiWord als Kathedrale, wurde aber eher zu einem Bazar. AbiWord ist Teil eines größeren Projekts, bekannt als AbiSource, welches von SourceGear Corporation begonnen wurde. Das Ziel dieses Projekts war das Entwickeln einer plattformübergreifenden Open Source Office Suite beginnend mit AbiWord, dem Textverarbeitungsprogramm des Projekts.

"SourceGear veröffentilchte den Source Code von AbiWord und schnell formte sich eine Entwicklergemeinde um das Projekt. SourceGear hat seitdem nicht mehr an dem Projekt gearbeitet."

"Seitdem setzt die Entwicklergemeinde Verbesserungen um und erhöht die Qualität von AbiWord. Version 1.0 wurde veröffentlicht im April 2002, gefolgt von Version 2.0 im September 2003, 2.2 im December 2004 und 2.4 im September 2005. " Zitat Ende

Die letzte stabile Version vom 02/07/2007 ist Version 2.4.6
