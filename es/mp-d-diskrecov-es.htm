Herramientas de Recuperación de Disco

Gparted

Por favor, vea el Manual Principal de Gparted

Para ver la documentación completa, vaya a GParted

MagicRescue

MagicRescue (RescateMágico) recupera archivos a través de la búsqueda de "bytes mágicos" con un dispositivo de bloque para ciertos tipos de archivos y lanza un programa externo para extraerlos. Ya que revisa los "bytes mágicos" contenidos en el archivo, puede servir como utilidad para des-borrar y también para recuperar un drive o partición corrompido. Mientras los datos del archivo existan puede encontrarlo.

Es importante leer la documentación completa de MagicRescue antes de usarlo.

TestDisk

TestDisk es un software poderoso y libre para la recuperación de datos! Fue diseñado primeramente para recuperar particiones y/o para convertir discos no arrancables en arrancables de nuevo, si el cambio era causado por fallo del software, algunos tipos de virus o error humano (tales como un borrando acidental de la Tabla de Particiones). Es sumamente fácil la recuperación de Tablas de Particiones con TestDisk.

TestDisk puede

  • Corregir la Tabla de Particiones. Recuperar una partición borrada
  • Recuperar el sector de arranque FAT32 de su respaldo
  • Reconstruir el sector de arranque FAT12/FAT16/FAT32
  • Reparar Tablas de FAT
  • Reconstruir el sector de arranque NTFS
  • Recuperar el sector de arranque NTFS de su respaldo
  • Reparar MFT utilzando el espejo de MFT
  • Buscar el SuperBlock de Respaldo de ext2/ext3

TestDisk tiene características tanto para expertos como para novatos. Para los que saben poco o nada acerca de recuperación de datos, puede ser usado para recoger información detallada acerca de un dispositivo que después puede ser enviada a un técnico para un mayor analisis. Aquellos que tienen familiaridad con estos procedimientos deben encontrar en TestDisk una herramienta muy potente para realizar una recuperación en la misma máquina que tiene el problema.
