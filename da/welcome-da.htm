Welcome to sidux™ Community Software Picks

This is a special manual designed to give you an overview on the packages recommended by some special interest groups in the sidux community.

It should save you hours af trawling through debian to find a genre of games or other suites of software to suit your hardware and needs.

You will probably need to add contrib/ non-free to your /etc/apt/sources.list.d/debian.list and /etc/apt/sources.list.d/sidux.list or /etc/apt/sources.list. Please refer to the sidux Operating System Manual Adding non-free to Sources List.

Welcome to sidux!

How to use this manual

This manual is divided into common sections that mirrors the main meta-installer list, for example, Education has links to the various components contained in the meta-package using common files.

As with any new documentation there will be errors/mistakes/typos, although we hope otherwise regarding errors and mistakes (typos we can live with ) please forgive us.

As the body of work grows, more documentation will be added and we at sidux are sure that this will prove to be a very valuable resource for you and we thank you for being a part of sidux.

Notice: The Descriptions were taken from Debian's Package Database using the apt-cache show command. The screenshots and hints/notes were all done by the sidux-community.
