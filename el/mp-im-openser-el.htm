OpenSIPS Project

Εικόνες

Περιγραφή

Το OpenSIPS (πριν OpenSER) είναι ένας πολύ γρήγορος και ευέλικτος SIP (RFC3261) proxy διακομιστής. Γραμμένο εξ' ολοκλήρου στην C, το OpenSIPS μπορεί να χειριστή χιλιάδες κλίσεις το δευτερόλεπτο ακόμη και σε φτηνό εξοπλισμό.

Ανάμεσα σε άλλα αυτά τα modules είναι διαθέσημα: digest authentication, CPL scripts, instant messaging, MySQL support, presence agent, RADIUS authentication, record routing, SMS gateway, Jabber/XMPP gateway, transaction module, registrar και user location, XML-RPC interface.

C Shell-παρόμοια γλώσσα scripting παρέχει πλήρη έλεγχο της συμπεριφοράς του διακομιστή. Η αρχιτεκτονική του σας επιτρέπει να φορτώσετε μόνο τις απαραίτητες για εσάς λειτουργίες.

Αυτό το πακέτο περιέχει το κυρίως OpenSIPS binary μαζί με ορισμένα modules και απαραίτητα binaries.
