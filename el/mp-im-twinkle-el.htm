Twinkle

Εικόνες

Περιγραφή

Το Twinkle είναι ένα Voice over Internet Protocol (VoIP) SIP τηλέφωνο λογισμικού για διενέργεια κλίσεων με χρήση SIP με ένα IP δίκτυο και υποστηρίζει απευθείας κλίσεις IP τηλέφωνο με IP τηλέφωνο ή με ενός δικτύου με SIP proxy για δρομολόγηση των κλίσεων σας.

Εκτός από τις βασικές κλίσης φωνής το Twinkle παρέχει και τις πιο κάτω λειτουργίες ασχέτως από τις υπηρεσίες που παρέχει ο VoIP πάροχος σας:

  • 2 γραμμές κλίσεων
  • Πολλαπλές ενεργές ταυτότητες κλίσεων
  • Ήχος κλίσης της επιλογής σας
  • Αναμονή κλίσης
  • Call Hold
  • Κλίσης συνεδρίασης με 3 άτομα
  • Mute
  • Ανακατεύθυνση κλίσης όταν εσείς το θέλετε
  • Γενική ανακατεύθυνση κλίσεων
  • Ανακατεύθυνση κλίσης όταν είναι κρατημένο
  • Ανακατεύθυνση κλίσης όταν δεν απαντηθεί το τηλέφωνο
  • Απόρριψη αίτησης ανακατεύθυνσης κλίσης
  • Τυφλή μεταφορά κλίσεων
  • Μεταφορά κλίσης με συμβούλευση (νέο)
  • Απόρριψη αίτησης μεταφοράς κλίσης
  • Απόρριψη κλίσης
  • Επανάληψη τελευταίας κλίσης
  • Μην ενοχλείτε
  • Αυτόματη απάντηση
  • Ένδειξη για μήνυμα σε αναμονή
  • Voice mail speed dial
  • Scripts γραμμένα από τον χρήστη τα οποία εκτελούνται με τις κλίσεις
  • Π.χ. για απόρριψη συγκεκριμένων κλίσεων
  • RFC 2833 DTMF events
  • In-band DTMF
  • Out-of-band DTMF (SIP INFO)
  • STUN υποστήριξη για NAT traversal
  • Αποστολή NAT keep alive πακέτων όταν χρησιμοποιείτε STUN
  • NAT traversal through static provisioning
  • Ένδειξη χαμένης κλίσης
  • Ιστορικό κλίσεων, λεπτομερή αρχείο για εισερχόμενες, εξερχόμενες, επιτυχημένες και χαμένες κλίσεις
  • DNS SRV υποστήριξη
  • Αυτόματη αλλαγή διακομιστή όταν ένας διακομιστής δεν είναι διαθέσιμος
  • Άλλα προγράμματα μπορούν να διενεργήσουν κλίσεις μέσω του Twinkle, π.χ. κλίσεις από την ατζέντα
  • System tray εικονίδιο
  • System tray μενού για διενέργεια κλίσεων όσο το Twinkle παραμένει κρυμμένο
  • Κανόνες μετατροπής αριθμών καθορισμένες από τον χρήστη
  • Απλή ατζέντα
  • Υποστήριξη για UDP και TCP (νέο) σαν μεταφορά για το SIP
  • Παρουσία
  • Instant messaging

VoIP ασφάλεια
  • Ασφαλή επικοινωνία φωνής με ZRTP/SRTP
  • MD5 digest authentication υποστήριξη για όλες τις SIP αιτήσεις
  • AKAv1-MD5 digest authentication υποστήριξη για όλες τις SIP αιτήσεις (νέο)
  • Απόκρυψη ταυτότητας
Codecs για ήχο
  • G.711 A-law (64 kbps payload, 8 kHz sampling rate)
  • G.711 u-law (64 kbps payload, 8 kHz sampling rate)
  • GSM (13 kbps payload, 8 kHz sampling rate)
  • Speex narrow band (15.2 kbps payload, 8 kHz sampling rate)
  • Speex wide band (28 kbps payload, 16 kHz sampling rate)
  • Speex ultra wide band (36 kbps payload, 32 kHz sampling rate)
  • G.726 (16, 24, 32 or 40 kbps payload, 8 kHz sampling rate)
Για όλα τα codecs οι παρακάτω επιλογές είναι διαθέσιμες για την βελτίωση της ποιότητας στο άλλο άκρο της "γραμμής".
  • Automatic gain control (AGC) (new)
  • Μείωση θορύβου (νέο)
  • Ανίχνευση δραστηριότητας φωνής (VAD) (νέο)
  • Acoustic echo control (AEC) [experimental] (νέο)
