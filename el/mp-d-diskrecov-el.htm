Εργαλεία Ανάκτησης (Recovery) Δίσκου

Gparted

Παρακαλώ αναφερθείτε στο κυρίως εγχειρίδιο

Για πλήρη τεκμηρίωση δείτε GParted

MagicRescue

Το magicrescue ανακτά αρχεία με το να ψάχνει μια block συσκευή για μαγικά (magic) bytes τύπων αρχείων τα οποία γνωρίζει και μπορεί να ανακτήσει και μετά καλεί ένα εξωτερικό πρόγραμμα για να τα αποσπάσει (extract). Κοιτάζει τα "magic bytes" του περιεχόμενου του αρχείου και έτσι μπορεί να χρησιμοποιηθεί και σαν εργαλείο ανάκτησης διαγραμμένων αρχείων και σαν εργαλείο ανάκτησης από κατεστραμμένους (corrupted) δίσκους ή κατατμήσεις (partition). Όσο τα αρχεία είναι εκεί, θα τα βρει.

Είναι σημαντικό να διαβάσετε την πλήρη τεκμηρίωση του magicrescue

TestDisk

Το TestDisk είναι ένα ισχυρό, ελεύθερο λογισμικό ανάκτησης δεδομένων! Σχεδιάστηκε με πρωταρχικό σκοπό να βοηθήσει στην ανάκτηση χαμένων κατατμήσεων ή/και να διορθώσει δίσκους οι οποίοι δεν εκκινούν όταν αυτό προκαλείται από ελαττωματικό λογισμικό, ορισμένους ιούς ή ανθρώπινο λάθος (όπως η από λάθος διαγραφή του Πίνακα Κατατμήσεων). Η ανάκτηση Πίνακα Κατατμήσεων είναι πολύ εύκολη με την χρήση του TestDisk.

Το TestDisk μπορεί

  • Να διορθώσει ένα πίνακα κατατμήσεων (partition table), να ανακτήσει διαγραμμένη κατάτμηση
  • Να ανακτήσει ένα FAT32 boot sector από ένα αντίγραφο ασφαλείας
  • Να ξαναδημιουργήσει FAT12/FAT16/FAT32 boot sector
  • Να διορθώσει FAT πίνακες
  • Να ξαναδημιουργήσει NTFS boot sector
  • Να ανακτήσει NTFS boot sector από ένα αντίγραφο ασφαλείας
  • Να διορθώσει MFT χρησιμοποιώντας MFT mirror
  • Να εντοπίσει ext2/ext3 Backup SuperBlock

Το TestDisk έχει χαρακτηριστικά (features) και για αρχάριους και για προχωρημένους. Για αυτούς που ξέρουν λίγα ή τίποτε από τεχνικές ανάκτησης δεδομένων, το TestDisk μπορεί να χρησιμοποιηθεί για την συλλογή λεπτομερών πληροφοριών για ένα δίσκο που δεν εκκινεί (non-booting) οι οποίες μπορούν μετά να σταλούν σε ένα τεχνικό για περαιτέρω ανάλυση. Αυτοί που είναι πιο εξοικειωμένοι με τέτοιες διαδικασίες θα δουν ότι το TestDisk είναι ένα εύχρηστο εργαλείο για επιτόπου ανάκτηση (recovery).
