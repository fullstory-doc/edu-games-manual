KOffice

A suíte KOffice inclui as seguintes aplicações:

Produtividade
  • KWord - Processador de texto que funciona em dois modos: orientado a página e orientado a leiaute
  • KSpread - Planilha eletrônica poderosa
  • KPresenter - Programa completo de apresentações
  • Kexi - Ambiente integrado para a criação de bancos de dados e aplicações referentes a bancos de dados
Criatividade
  • Kivio - Diagramas no estilo Visio®
  • Karbon14 - Desenho vetorial
  • Krita - Manipulação de imagens
Gerenciamento
  • KPlato - Ferramenta que integra gerenciamento de projetos e planificação
Suporte
  • KChart - Ferramenta para desenhar gráficos e tabelas
  • KFormula - Poderoso editor de fórmulas
  • Kugar - Ferramenta para gerar relatórios comerciais com qualidade

Openoffice

OpenOffice.org 2 - Descrição

Compatível com várias outras suítes de escritório

  • Writer – processador de textos que você pode usar para tudo, desde escrever uma cartinha até um livro inteiro
  • Calc – poderosa planilha com todas as ferramentas necessárias para você calcular, analisar e apresentar dados em relatórios numéricos ou em gráficos espetaculares
  • Impress – a maneira mais rápida e poderosa para você criar apresentações multimídia
  • Draw – ferramenta de gráficos vetoriais para você criar diagramas simples ou ilustrações 3D
  • Base – banco de dados para criação ou modificação de tabelas, formulários, questionários e relatórios, tudo dentro do OpenOffice.org
  • Math – aplicação onde você pode criar equações matemáticas tanto numa interface gráfica quanto digitando suas fórmulas diretamente no editor

Gnumeric

A planilha Gnumeric faz parte do amigável ambiente gráfico Gnome.

O objetivo do Gnumeric é ser a melhor planilha possível. Os desenvolvedores não tentam clonar aplicações já existentes. Todavia, o Gnumeric lê arquivos salvos em outras planilhas e torna fácil o período de transição necessário para você se adaptar com ele.

AbiWord

Vamos citar o que está no site do Abiword: "Como muitos outros projetos de Código Aberto, o Abiword começou como uma catedral, mas hoje se parece mais com um bazar. O Abiword é parte de um projeto maior, conhecido por AbiSource, iniciado pela SourceGear Corporation. O objetivo do projeto era desenvolver uma suíte de escritório multiplataforma de Código Aberto, começando com o Abiword, o processador de textos do projeto".

"A SourceGear disponibilizou o código fonte do Abiword e logo se formou uma comunidade de desenvolvedores em torno dele. Então, a SourceGear deixou de trabalhar no projeto."

"Os desenvolvedores continuaram a fazer melhorias e aumentar a qualidade do Abiword. A versão 1.0 saiu em Abril de 2002, seguida da versão 2.0 em Setembro de 2003, 2.2 em Dezembro de 2004 e 2.4 em Setembro de 2005."

A mais recente versão estável é a 2.4.6, lançada em 02/07/2007.
