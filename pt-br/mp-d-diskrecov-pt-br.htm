MagicRescue

O MagicRescue recupera arquivos procurando "bytes mágicos" referentes aos tipos de arquivos que ele sabe como recuperar. Daí, ele chama um programa externo para extraí-los. Ele olha os "bytes mágicos" no conteúdo do arquivo, de forma que pode ser usado tanto na função 'undelete' (desapagar) quanto na função de recuperar um HD ou partição corrompidos. Desde que os dados estejam ali, eles os encontrará.

>É de suma importância ler toda a documentação do programa.

TestDisk

O TestDisk é um poderoso software livre para recuperação de dados! Ele foi planejado principalmente para ajudar na recuperação de partições apagadas e/ou fazer com que HDs avariados sejam capazes de dar o boot novamente, quando a causa tiver sido softwares defeituosos, certos tipos de vírus ou erro humano (como apagar sem querer a tabela de partições). Recuperar a tabela de partições com o TestDisk é realmente muito fácil.

O TestDisk pode:

  • Consertar a tabela de partições e recuperar partições apagadas
  • Recuperar o setor de boot FAT32 a partir de seu backup
  • Reconstruir o setor de boot FAT12/FAT16/FAT32
  • Consertar tabelas FAT
  • Reconstruir o setor de boot NTFS
  • Recuperar o setor de boot NTFS a partir de seu backup
  • Consertar MFT usando espelho MFT
  • Localizar o SuperBloco de backup ext2/ext3

O TestDisk pode ser usado tanto por iniciantes quanto por usuários experientes. Para aqueles que nada sabem das técnicas de recuperação, o TestDisk pode ser usado para coletar informações detalhadas sobre um HD que não dá boot mais. Esses dados, então, podem ser entregues a um técnico para posterior análise. Já aqueles que estiverem familiarizados com tais procedimentos encontrarão no TestDisk a ferramenta ideal para recuperação por conta própria.
