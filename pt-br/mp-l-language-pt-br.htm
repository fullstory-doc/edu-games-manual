Pacotes de Linguagem

Croata

Há três meta-pacotes:

  1. Croatian Additional - instala configurações do ambiente para linguagens e o guia dos mantenedores do Debian; pacotes: language-env e maint-guide
  2. Croatian - instala o ambiente croata (KDE, páginas de manual, ortografia); pacotes: kde-i18n-hr, aspell-hr, myspell-hr
  3. Croatian Openoffice - OpenOffice para os croatas; pacotes: openoffice.org-l10n-hr, openoffice.org-hyphenation-hr

Polonês

Há três meta-pacotes

  1. Language-Polish_Core - instala o ambiente polonês (KDE, páginas de manual, ortografia); pacotes: kde-i18n-pl, aspell-pl, manpages-pl, myspell-pl, ipolish, k3b-i18n
  2. Language-Polish_Openoffice - Openoffice em polonês; pacotes: openoffice.org-l10n-pl, openoffice.org-hyphenation-pl, openoffice.org-thesaurus-pl
  3. Language-Polish_Icedove_Iceweasel - Icedove e Iceweasel em polonês; pacotes: Icedove-locale-pl, iceweasel-l10n-pl

Russo

Há quatro meta-pacotes

Pacote 1
  • apt-howto-ru - guia com exemplos do APT em russo
  • aspell-ru - dicionário russo para o Aspell
  • childsplay-alphabet-sounds-ru - arquivos de som em russo para o jogo Childsplay
  • console-cyrillic - suporte melhorado ao cirílico
  • dict-freedict-eng-rus - dicionário Inglês/Russo Freedict
  • easytag - ver, editar e escrever etiquetas ID3
  • emacs-intl-fonts - fontes Postscript que podem ser impressas no Emacs
  • enigmail-locale-ru - pacote de linguagem russa para o Enigmail
Pacote 2
  • firefox-locale-ru - pacote de linguagem russa para o Firefox
  • gcompris-sound-ru - arquivos de som em russo para o GCompris
  • icedove-locale-ru - pacote de linguagem russa para o IceDove
  • iceweasel-l10n-ru - pacote de linguagem russa para o Iceweasel
  • irussian - dicionário russo para o Ispell
  • kde-i18n-ru - arquivos de internacionalização em russo para o KDE
  • keyboards-rg - vários leiautes de teclado para o X e o console
  • kicad-doc-ru - arquivos de ajuda em russo para o Kicad
  • koffice-i18n-ru - internacionalização em russo para o KOffice
  • konwert - pacote com conversão de caracteres para arquivos ou terminal I/O
  • language-env - ferramenta de configuração simples para o ambiente da língua nativa
  • maint-guide-ru - tradução para o russo do Debian New Maintainers Guide (Novo Guia dos Mantenedores do Debian)
  • manpages-ru - tradução para o russo das Páginas de Manual
  • mozilla-firefox-locale-ru - pacote de linguagem russa para o Iceweasel (pacote de transição)
  • myspell-ru - dicionário russo para o MySpell
  • openoffice.org-help-ru - arquivo de ajuda em russo para o OpenOffice.org
  • openoffice.org-l10n-ru - pacote de linguagem em russo para o OpenOffice.org
  • python-pytils - bibliotecas Python para processamento de strings em russo
  • Russian Anywhere - decodificador de caracteres russos
  • sylpheed-claws-gtk2-i18n - localização para o Sylpheed-Claws GTK2 (suporte i18n)
  • sylpheed-claws-i18n - localização para o Sylpheed Claws (suporte i18n)

Greek

There are two packages:

  1. Greek - Installs the Greek environment for KDE, the Greek KDE manual, two spell checking applications for greek and the Greek sidux manual - packages: kde-i18n-el, aspell-el, myspell-el-gr, sidux-manual-el
  2. Greek Openoffice - Greek environment for OpenOffice - packages: openoffice.org-l10n-el

Portugese

There are two packages:

  1. Portugese - Installs the Portugese environment for KDE, the Portugese KDE manual, two spell checking applications for Portugese and the Portugese (pt-br) sidux manual - packages: kde-l10n-ptbr , aspell-pt-br , myspell-pt-br , sidux-manual-pt-br
  2. Portugese Openoffice - Portugese environment for OpenOffice - packages: openoffice.org-l10n-pt-br
